# Jelly Stock (젤리주식) 웹 프로젝트

이 프로젝트는 주식 시장의 구조를 초보자도 부담 없이 이해할 수 있도록,
현실적인 주식 데이터와 가상의 젤리 종목을 결합하여 구현한 React 기반 웹 서비스입니다.

실제 주식 웹사이트는 초보자에게 어려움과 두려움을 줄 수 있습니다.
정보량이 많고 처음 보는 용어들이 많기 때문에 공부를 시작하기가 쉽지 않습니다.
그래서 본 프로젝트는 "젤리"라는 친숙한 테마를 이용하여,
주식의 기본적인 흐름과 구조를 재미있고 가볍게 경험할 수 있도록 기획하였습니다.

원래는 "젤리나라 쫀즙마을" 같은 세계관의 재미있는 젤리 웹페이지를 만들고 싶었지만,
과제 요구사항이 API 활용이었기 때문에,
실제 한국 주식 시세 API와 젤리 테마의 가상 종목을 결합해 “재미있는 가상 투자 서비스”로 재구성했습니다.

그 결과,

- 실제 주식 데이터 조회(공공데이터포털 API),
- 랜덤 변동 기반 젤리 가상 종목 마켓,
- 지갑 및 충전 시스템,
- 간단한 인증 시스템,
- 무료 AI API를 활용한 투자 기초 안내 챗봇
  등 다양한 기능을 포함한 웹 서비스가 완성되었습니다.

---

## 1. 주요 기능

### 1) 대시보드 (Dashboard)

- 공공데이터포털 API를 통해 실제 주식 시세 조회
- 검색어 기반 종목 필터링
- 최신 날짜 기준 데이터 정렬
- 등락률 계산 및 색상 표현

### 2) 젤리 마켓 (Jelly Market)

- 딸기젤리, 포도젤리 등 가상 젤리 종목 제공
- 랜덤 변화 알고리즘 적용 (실시간처럼 보이는 가격 변동)
- 카드형 / 테이블형 UI 지원
- 가격 히스토리를 기반으로 한 작은 미니 그래프 표시
- 가상 매수 / 매도 기능 구현

### 3) 지갑 및 충전 기능 (Wallet)

- localStorage 기반 지갑 데이터 저장
- 현금과 젤리 보유량 표시
- 1회 충전 한도 및 하루 충전 한도 적용
- 충전 기록 및 젤리 전환 내역 저장

### 4) 회원가입 및 로그인 (Auth)

- localStorage 기반 간단한 인증 구조
- 로그인 후 Wallet, MyPage 접근 가능
- 세션 유지 기능 포함

### 5) AI 챗봇 (JellyBot)

- 무료 AI API 활용 (API Key 불필요)
- 주식, ETF, 투자 기초 개념을 쉬운 설명으로 제공
- 대화 초기화 기능 포함
- 사이트 어디에서나 접근 가능

## 2. 기술 스택

- React 18
- JavaScript (ES6+)
- CSS 모듈 단위 스타일 구성
- 무료 AI API (Key 없이 사용 가능)
- 공공데이터포털 한국 주식 시세 API
- LocalStorage를 활용한 데이터 보존
- 배포: Vercel 또는 Netlify

## 5. 주의 사항

- 로그인 및 회원가입 기능은 학습용(localStorage 기반)으로 제작되었습니다.
- 지갑 및 계정 데이터는 모두 브라우저 localStorage에 저장됩니다.
- 무료 AI API의 특성상 응답 속도가 느릴 수 있으나 정상 동작 범위입니다.
- 외부 API(공공데이터포털)의 경우 데이터 응답 속도에 따라 딜레이가 발생할 수 있습니다.

---

## 6. 프로젝트 목적

이 프로젝트는 다음과 같은 학습 및 기획 목표를 기반으로 제작되었습니다.

1. 초보자도 주식 시장의 구조를 이해할 수 있는 쉬운 UI 구현
2. 부담을 줄이기 위해 주식을 "젤리" 테마로 재해석
3. React 기반 컴포넌트 구조 및 상태 관리 학습
4. 외부 API 연동 및 비동기 데이터 처리 경험
5. LocalStorage 기반 클라이언트 단 인증 및 데이터 보존
6. 웹 프로젝트에서 AI 기능 적용 경험
7. 과제 요구사항(API 사용)을 충족하면서도 창의적인 기획을 반영

---

## 7. 결론

"Jelly Stock"은 주식 시세 조회, 가상 젤리 마켓, 지갑 관리, 인증, AI 챗봇까지
다양한 기능을 통합하여 제작한 React 기반 웹 프로젝트입니다.

초보자도 부담 없이 주식 개념을 체험해볼 수 있도록 기획했으며,
주식의 복잡함을 줄이고 친근한 테마로 구성하여
즐겁고 가볍게 이해할 수 있는 웹 서비스를 목표로 제작했습니다.
